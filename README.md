Prism-linuxtheme
====================

Dark glass linux theme

In this repository:
 - Aurorae theme (Window decorations)
   -> ~/.local/share/aurorae
 
 - Kvantum (QT5 theme)
   -> ~/.config/Kvantum
 
 - Plasma Desktop theme
   -> ~/.local/share/plasma/desktoptheme/
 
 - Plasma Splashscreen
